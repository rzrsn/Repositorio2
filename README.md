# Aplicación de Gestión de Tareas

## Descripción

Este proyecto es una aplicación web sencilla que permite gestionar tareas diarias. Su propósito es demostrar el uso de JavaScript para manipulación del DOM y almacenamiento local en el navegador.

## Funcionalidades

* Agregar nuevas tareas
* Marcar tareas como completadas
* Eliminar tareas
* Persistencia de datos mediante LocalStorage

## Arquitectura

La aplicación es completamente del lado del cliente:

* HTML: estructura de la interfaz
* JavaScript: lógica y manejo de eventos
* LocalStorage: almacenamiento de datos

## Tecnologías

* HTML5
* JavaScript

## Ejecución

### Clonar repositorio

```bash
git clone https://github.com/rzrsn/Repositorio2.git
```

### Ejecutar

Abrir el archivo `index.html` en cualquier navegador.
